Dashboard Fornecedor

Estado: Aberto (250px)

Bem-vindo ao Dashboard

Este é um teste visual da implementação do sidebar com toggle open/close. Clique no botão "Toggle Sidebar" ou use o checkbox para alternar entre os estados:

  • Aberto: Sidebar com 250px de largura mostrando ícone + texto
  • Fechado: Sidebar com 70px de largura mostrando apenas ícones
  • Passe o mouse sobre os itens para ver o efeito gradiente laranja
  • O conteúdo principal se adapta quando o sidebar abre/fecha

Funcionalidades Implementadas

✓ Sidebar responsivo com toggle open/close
✓ Transições suaves com CSS (0.3s ease)
✓ Gradient hovers em tons de laranja/azul
✓ Adapta automaticamente no mobile (off-screen)
✓ Persiste preferência do usuário em localStorage
✓ Efeito visual ao passar mouse nos itens
